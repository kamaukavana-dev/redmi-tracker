## System Overview

Redmi Tracker is a production-grade location intelligence platform that provides:

- Real-time GPS tracking with zero data loss
- State-driven geofencing with transition-based alerts
- Advanced analytics (speed, distance, anomaly detection)
- Device health monitoring and battery prediction
- Telegram notifications with retry logic
- Comprehensive observability

### Key Characteristics

- **Deterministic:** Same input вҶ’ same output (verified by 189 tests)
- **Testable:** 100% test coverage for critical paths
- **Observable:** Structured logging, correlation IDs, decision tracking
- **Maintainable:** Clean architecture, documented, no dead code
- **Secure:** API key authentication, constant-time comparison, input sanitization
- **Resilient:** Graceful degradation, retry logic, exception handling

5. **Sanitization:** Flexible type conversion
6. **Bounds Checking:** Coordinate/battery ranges

## Failure Modes

### Identified and Mitigated Risks

| Failure Mode | Impact | Mitigation | Status |
|--------------|--------|------------|--------|
| Invalid JSON payload | Data loss | Accept with "invalid" quality | вң… Fixed |
| Missing coordinates | Geofence skip | Accept with "degraded" quality | вң… Fixed |
| String coordinates | 422 errors | Flexible parsing | вң… Fixed |
| Empty string values | Validation fail | Convert to null | вң… Fixed |
| Null values | 422 errors | Handled gracefully | вң… Fixed |
| GPS drift | Alert spam | Cooldown + state machine | вң… Fixed |
| Boundary oscillation | Duplicate alerts | State transitions only | вң… Fixed |
| Telegram outage | Missed alerts | Retry x3 + graceful degradation | вң… Fixed |
| Database connection loss | Service failure | Pool pre-ping + recycle | вң… Mitigated |
| Scheduler overlap | Duplicate work | max_instances=1 | вң… Fixed |
| Clock skew | Incorrect state | Server timestamp authority | вң… Mitigated |
| Rate limit bypass | DoS risk | Per-key limiting | вң… Mitigated |
| Memory leak | Crash | No global state accumulation | вң… Verified |
| Silent data loss | Missing records | Raw payload preservation | вң… Fixed |
